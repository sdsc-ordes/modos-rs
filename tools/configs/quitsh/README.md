# Quitsh Configuration Files

These are some files which are used as `--config` argument on `quitsh`
invocations.

# User Settings

You can place here an ignored `config.user.yaml` which any `quitsh` invocation
will take up since `QUITSH_CONFIG_USER` is set to this file.

# CI Settings

The file `config-ci.yaml` is set as `QUITSH_CONFIG` in CI.
